1. FULL INSTALL — ONE UAC PROMPT
--------------------------------
Open a normal PowerShell in the WinYOLO project directory and run exactly:

Set-ExecutionPolicy -Scope Process Bypass -Force
& .\scripts\install.ps1 -Full

Accept the single UAC prompt. The elevated continuation installs/updates Bun,
Codex, Git, .NET 8 SDK, Visual Studio Build Tools/MSBuild, and Windows SDK;
installs WinYOLO and its plugin; creates/reuses WinYOLORunner; stores a new
runner password encrypted with Windows DPAPI; and applies protected ACLs.

7. ROLLBACK PROCEDURE
---------------------
winyolo checkpoint diff cp-REPLACE-WITH-ID
winyolo checkpoint rollback cp-REPLACE-WITH-ID

Rollback always retains:
%LOCALAPPDATA%\WinYOLO\workspaces\<run-id>\result.patch

Accept instead:
winyolo checkpoint accept cp-REPLACE-WITH-ID

Accept refuses if the source repository changed after checkpoint creation.
Reconcile current changes, inspect the exported patch, and retry deliberately.
