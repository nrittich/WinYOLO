WinYOLO — Windows Quick Start
==============================

Project folder
--------------
C:\Users\NickT\AI\Project\WinYOLO

Open PowerShell and run:

  cd C:\Users\NickT\AI\Project\WinYOLO

If PowerShell says Bun is not recognized
----------------------------------------

The PowerShell window was probably opened before Bun was installed. Either
close it and open a new PowerShell window, or refresh PATH in the current one:

  $env:Path = [Environment]::GetEnvironmentVariable("Path", "Machine") + ";" + [Environment]::GetEnvironmentVariable("Path", "User")
  bun --version

Expected result:

  1.3.14

Absolute-path fallback:

  & "$env:USERPROFILE\.bun\bin\bun.exe" --version

Choose one model path
---------------------

Option A — Codex planner

  codex login
  .\winyolo.cmd serve

Option B — GPT-5.6 Responses API

  $env:OPENAI_API_KEY = "your-openai-api-key"
  .\winyolo.cmd serve

Then open this address in your browser:

  http://127.0.0.1:4747

Useful commands
---------------

  .\winyolo.cmd doctor
  .\winyolo.cmd demo
  powershell -NoProfile -ExecutionPolicy Bypass -File scripts\smoke-windows.ps1
